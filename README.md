## Cheats

| Código     | Acción                                                             |
|------------|--------------------------------------------------------------------|
| STRONGMIND | Ningún ataque hace daño al jugador                                 |

| CANDY      | Obtener pistola de rayos (RayGun).                                 |
| STICK      | Obtener espada (ShockSword).                                       |
| ATODOGAS   | Super velocidad del jugador.                                       |
| AEGIS      | Pasar al siguiente nivel.                                          |
